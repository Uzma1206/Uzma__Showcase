# 🎞️ Uzma Memories / Uzma Showcase

🌐 **Live Demo:** https://uzma-showcase.vercel.app

A Netflix-inspired personalized memory-sharing platform designed to preserve and showcase meaningful moments through photos, videos, music, and interactive galleries.

Built using React, Firebase, Tailwind CSS, Cloudinary, and Vercel, the platform provides a secure and immersive experience for storing and reliving memories.

---

## ✨ Key Features

- 🔐 Secure Firebase Authentication
- 📸 Photo Gallery with Fullscreen Viewer
- 🎥 Video Gallery & Media Streaming
- 🎵 Background Music Controls
- ☁️ Cloudinary Media Storage
- 📱 Fully Responsive Design
- ⚙️ Admin Dashboard for Content Management
- 🚀 Fast Deployment with Vercel
- 🔄 Scalable Multi-Client Architecture
